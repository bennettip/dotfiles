dotfiles
===============

ONE-STEP INSTALL
----------------

Use curl:

```sh
curl -o - https://raw.githubusercontent.com/bennettip/dotfiles/master/auto-install.sh | sh
```

or wget:

```sh
wget -O - https://raw.githubusercontent.com/bennettip/dotfiles/master/auto-install.sh | sh
```
